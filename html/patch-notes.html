Patch Notes

Recent updates, fixes, and additions

patch note 0.4:

UPDATED: Game layout now adapts better across browsers and screen sizes, with board and hand areas staying inside their borders more reliably;

FIXED: Banana Onemoglosti now negates card effects, keywords, and continuous effects until the end of the turn;

FIXED: Hvala hvala hvala can now be played during counter phase;

UPDATED: Mr. Jeremič now lets you type a card name, plays a matching card from the deck if found, then shuffles the deck;

FIXED: Johan Johanovič now places both chosen cards on top of the deck or both on bottom;

UPDATED: Stages can no longer be played over another stage;

patch note 0.3:

ADDED: Deck Editor can now save decks locally, load them later, delete them, and copy/import deck text;

UPDATED: Local deck saves use browser local storage only, so decks stay on your current browser/device and are not uploaded online;

UPDATED: Saved deck data includes deck name, selected leader, and card list, while deck text uses the same format as preset decks like `4xCARD-ID`;

HOW-TO: To build a deck, choose a Leader first, add cards, then save or copy the deck text if needed;

HOW-TO: In VS Self, each player deck is chosen from a popup where you can use a pasted deck, a preset deck, or a locally saved deck;

HOW-TO: For multiplayer, deck selection is now done only on the Queue page before creating or joining a room;

patch note 0.2:

ADDED: Multiplayer turn setup now follows dice roll, turn order choice, and mulligan;

ADDED: Live multiplayer board syncing;

UPDATED: Each player now always sees their own board on bottom;

UPDATED: Opponent hand, life, and deck stay hidden in multiplayer, while trash stays visible;

patch note 0.21:

FIXED: Queue page now lets players choose a deck before entering multiplayer;

FIXED: Match setup no longer gets stuck on preparing match and now starts from dice roll;

FIXED: Starting hand is dealt after turn order is chosen;

FIXED: Turn passing, battle log pregame entries, and defender-only blocker controls now work correctly;

FIXED: Blocker phase can be skipped, and defender can resolve during counter phase;

patch note 0.22:

UPDATED: Game behaviour settings now apply directly to singleplayer and multiplayer gameplay;

ADDED: Auto draw now also handles DON!! phase automatically when enabled;

ADDED: End turn, counter, and trigger confirmations now follow saved settings;

ADDED: Auto skip block, auto skip trigger, and auto select max value now affect in-game choices;

patch note 0.1:

ADDED: Patch Notes page on main menu;
